vue-element-admin-template

## Project setup

```
yarn install
```

### Touch .env.local

```bash
NODE_ENV=环境mode
VUE_APP_TOKEN=token存储的键值
VUE_APP_EMAIL=联系邮箱地址
VUE_APP_DOMAIN=cookie存储的域名
VUE_APP_API=api地址
VUE_APP_WEB=web跳转地址
```
